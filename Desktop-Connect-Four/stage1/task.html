Description

When you want to play a game, the first step is setting up the board! What exactly is a game board? It can be represented in a lot of different ways. For our game of Connect Four, it is a set of rows and columns that intersect each other, forming individual cells that can each store a game piece. The rows and columns help us keep the pieces organized so that we can easily see when a player has linked four in a row. You can see an example below:

             
             
        O    
      O X    
    O X X O  
O O X X O X X

We will use just such a board to play our game. At this stage, we won't concern ourselves with the pieces but with creating an empty board with nicely aligned rows and columns ready for players to use. We will label these columns by number to make it easier for players to correctly place their pieces later.

Objectives

Your program's output should be an empty game board as described below:

  1. There should be 6 rows and 7 columns for the part of the board where we can place pieces later.
  2. Use the character '_' in all the cells to create your empty board.
  3. There should be an additional row on top to label the columns by number from 1-7.
  4. Put spaces between all numbers and underscore characters.
