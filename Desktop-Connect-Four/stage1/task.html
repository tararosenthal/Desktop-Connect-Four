Description

When you want to play a game, the first step is setting up the board! What exactly is a game board? It can be represented in a lot of different ways. For our game of Connect Four, it is a set of rows and columns that intersect each other, forming individual cells that can each store a game piece. The rows and columns help us keep the pieces organized so that we can easily see when a player has linked four in a row. You can see an example below:

             
             
        O    
      O X    
    O X X O  
O O X X O X X

We will use just such a board to play our game. At this stage, we won't concern ourselves with the pieces but with creating an empty board with nicely aligned rows and columns ready for players to use. We will label the columns by uppercase letter and the rows by number to make it easier to understand how the board should be organized.

Objectives

Your program should display an empty game board as described below:

  1. The main class Connect4 should extend the class JFrame and exit on close.
  2. The cells should extend the class JButton. (Set focus painted to false to remove highlighting on clicked cells.)
  3. There should be 6 rows and 7 columns for the cells on the board.
  4. Each cell should be visible and display its row and column as its label (A1 - G6).
  5. Each cell should be named Button** where ** designates the same row and column designation as its label.
