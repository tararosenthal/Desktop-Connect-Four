Description

Now an empty game board isn't much use if we can't do anything with it. At this stage we will add two important features. The first is allowing the player to place game pieces. For the game of Connect Four, pieces are placed by column and then they fall down to rest either on the uppermost game piece in that column or, if it is empty, on the bottom of the board. So we will take a simple user input of the column number to place each game piece. We will only have one player for this stage, so we only need one type of game piece. The second feature we will add is allowing multiple input until the end of the game. In this one player version, the game can only end when the board is full, so we will keep the game going until then!

Objectives

In this stage, your program should:

  1. First output the empty game board and request user input to choose a column number.
  2. Place a game piece, in this case the letter 'O', in the chosen column and print the updated game board.
  3. Continue to request user input and print the result until the board is full.
  4. Finally, print the words, "Game over!"
